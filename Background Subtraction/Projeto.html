Problema: Detecção de objetos em movimento em vídeo¶

Dada uma stream de vdeo usar tcnicas de processamento de imagens para detectar objetos em movimento segmentando-os do plano de fundo e selecionando potenciais objetos de interesse. A soluo poder ser utilizada em vdeos de segurana para acionar a gravao da cena ou em vdeos de "homecare", onde o monitoramento de vulnerveis poder alertar sobre situaes de emergncia. As tcnicas utilizadas sero as vistas em sala de aula acrescidas de tcnicas especficas para vdeos enquanto sequencias de imagens.

Dados¶

Dataset de teste para problemas de extrao e modelagem de plano de fundo obtido em www.svcl.ucsd.edu/projects/background_subtraction/ucsdbgsub_dataset.htm.

O projeto faz uso de videos com camera fixa de pedestres e veiculos em transito, simulando uma camera de segurana em situao real.

from os.path import isfile, join
 
def convert_frames_to_video(pathIn,pathOut,fps):
    frame_array = []
    files = [f for f in os.listdir(pathIn) if isfile(join(pathIn, f))]
 
    #for sorting the file names properly
    files.sort(key = lambda x: int(x[6:-4]))
 
    for i in range(len(files)):
        filename=pathIn + files[i]
        #reading each files
        img = cv2.imread(filename)
        height, width, layers = img.shape
        size = (width,height)
        print(filename)
        #inserting the frames into an image array
        frame_array.append(img)
 
    out = cv2.VideoWriter(pathOut,cv2.VideoWriter_fourcc(*'DIVX'), fps, size)
 
    for i in range(len(frame_array)):
        # writing to a image array
        out.write(frame_array[i])
    out.release()

Extração de Plano de Fundo¶

Tcnica que consiste na separao dos objetos dinmicos (em movimento) dos objetos estticos em uma sequncia de imagens temporalmente correlacionadas. O principal desafio criar uma forma de deteco que seja robusta variaes de iluminao e eficiente do ponto de vista computacional. Geralmente os algoritmos proposto se baseam na criao de um modelo de plano de fundo, baseado nos frames vistos anteriormente, o qual ser usado para comparao com o frame observado. Dessa forma seguem o seguinte pipeline:

- Inicializao do modelo de plano de fundo at um certo limiar
- Manuteno do modelo de plano de fundo
- Deteco dos objetos em primeiro plano

Biblioteca BGS¶

O OpenCV apresenta hoje apenas dois algoritmo para extrao de plano de fundo nativamente, as outras opes que apresentava em verses anteriores foram movidas para a exteno da biblioteca conhecida como "Contrib", por essa razo optou-se pelo uso de implementaes suplementares disponveis em https://github.com/andrewssobral/bgslibrary implementadas pelo pesquisador Andrew Sobral e disponibilizada gratuitamente para uso academico e comercial. As implementaes esto na forma de um framework sobre o OpenCV com uma API relativamente simples e homognea para os diversos algoritmos. A biblioteca disponibilza hoje 43 implementaes de algoritmos e a forma mais simples de us-la inclu-la em projeto OpenCV em C++, portanto o cdigo do projeto implementado em C++ est incluido em anexo.

Rastreador de objetos¶

Baseando-se nos objetos encontrados pelo algoritmo LBMOG da biblioteca BGS foi implementado um rastreador de objetos em movimento em C++ que segue o seguinte pipeline.

- Obter um frame da camera ou vdeo
- Suavizar ruidos do frame com um filtro gaussiano
- Aplicar a subtrao de plano de fundo para obter uma mascara do primeiro plano
- Diminuir ruidos e fechar buracos na mascara com morfologia matemtica
- Encontrar contornos na mascara
- Selecionar os contornos que atendam criterios de interesse
- Criar "retangulos delimitadores" para os contornos
- Mostrar os resultados

Conclusão¶

A extrao de plano de fundo pode ser uma tcnica no inicio do processamento muito til em diversas aplicaes. A disponibilidade de uma biblioteca diversa, prtica e gratuita como a BGS muito interessante para o campo de Viso Computacional e este trabalho procura apresentar uma forma simples de utiliz-la em um contexto de vigilncia. A aplicao de rastreamento pode ser extendida de varias maneiras, incluindo formas mais robustas de manter o rastreamento, classificao dos objetos encontrados, contagem, "triggers" em regies de interesse, entre outras.
